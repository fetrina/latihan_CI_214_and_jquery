==== Latihan Buku CI & Jquery ====

------ Step By Step menggunakan Codeigniter pertama kali ---------

-Konfigurasi config.php 
Pada folder application/config, dlmnya ubah base_url dgn localhost/nama_foldermu
Misal jadi -> $config['base_url'] = 'http://localhost/latihan_CI_214_jquery'

-konfigurasi database
setting db hostname = localhost, username = root

-Latihan bikin controller --> ex: controler belajar.php
-Latihan bikin View --> ex: controler buku.php , viewnya buku1.php dan penulis2.php
-Latihan MVC tanpa database --> ex: model  mkendaraan.php, controler ckendaraan.php, view vkendaraan.php

-konfigurasi autoload.php
Di folder apliaction/config, ubah jadi
$autoload['libraries'] = array();
$autoload['helper'] = array();

-Konfigurasi database.php untuk MVC, 
Di folder application/config, databasenya isi dgn nama databasemu. Misal
$db['default']['database'] = 'latihan_cijquery';
$db['default']['dbdriver'] = 'mysql';

-Latihan bikin MVC pake database, Coba jalanin controller di browser. 
Klo ada error maslah enkripsi, brti ada yg musti diedit di config/config.php
Misal isi yg didalam tanda petik satu disini --> $config['encryption_key'] = '';


------- Contoh MVC dengan Database --------

- Bikin MVC untuk insert data. Filenya: mkomentar, ckomentar, tambahkomentar
- Bikin MVC untuk insert, view, update data. Filenya: mkomentar2, ckomentar2, tambahkomentar2, vkomentar2, ubahkomentar2
